How to deploy (Vercel)
----------------------
1) Go to https://vercel.com → Add New → Project
2) Click "Upload" and select this folder (or the ZIP).
3) Deploy. Done. You get a live URL immediately.

Custom domain
-------------
Add your domain in Vercel → Project → Settings → Domains and follow the DNS instructions.

Contact form
------------
The form is front-end only. Connect it to email using Vercel Serverless, a webhook, Power Automate, or a service like Formspree.
